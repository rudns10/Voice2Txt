# Voice2Txt 사용설명서

음성을 **텍스트로 바꿔주는** Windows 프로그램입니다. 녹음하거나 오디오 파일을 불러와 변환하고, 결과를 재생·편집·저장할 수 있어요. **모든 처리는 내 PC에서(오프라인)** 이뤄집니다.

---

## 1. 설치 / 실행

1. 받은 **`Voice2Txt-…-win-x64.zip`** 의 압축을 풉니다. (버전 숫자는 다를 수 있어요)
2. 압축 푼 폴더 안의 **`Voice2Txt.Gui.exe`** 더블클릭 → 실행.
3. 별도 설치 없음. (자주 쓰면 exe 우클릭 → 작업표시줄에 고정 추천)

> **첫 실행 시 파란 경고창(SmartScreen)** → **"추가 정보" → "실행"**. (서명 안 된 사내 프로그램이라 그래요. 안전합니다.)

- 대상: Windows 10/11 (64비트)
- 인터넷: **첫 변환 때 모델(약 190MB~)을 1회 다운로드**할 때만 필요. 이후 전부 오프라인.
  - 인터넷이 안 되는 **폐쇄망 PC**라면 모델이 동봉된 배포본(`…-with-small.zip` 등)을 받아야 합니다.

---

## 2. 화면 구성

- **왼쪽(사이드바)**: 녹음 목록 · 검색 · 맨 아래 **실시간 자막 토글**과 **변환 모델** 선택
  - "녹음 목록" 옆 **☑ 아이콘** = 여러 개 선택 삭제(선택 모드)
- **오른쪽**: 상황에 따라 바뀌는 작업 영역(녹음 / 변환 중 / 결과)
- **오른쪽 위 ☀️**: 테마(시스템 / 라이트 / 다크) 변경

---

## 3. 녹음하기

1. 가운데 **빨간 마이크 버튼** 클릭 (또는 **스페이스바**)
2. 녹음 중 **타이머·파형** 표시
   - **일시정지 / 재개**: [일시정지] 버튼 또는 **스페이스바**
   - **정지**: [정지] 버튼
3. 정지하면 **저장 창**이 뜹니다.
   - **파일 이름** 입력, **저장 위치** 선택
   - **"지금 텍스트로 변환"** 체크 시 저장 후 바로 변환
   - [저장] → 왼쪽 목록에 추가

> 1초 미만으로 너무 짧으면 저장되지 않습니다.

---

## 4. 오디오 파일 불러오기

이미 가진 음성 파일도 변환할 수 있어요. 두 가지 방법:

- **드래그 & 드롭**: 파일(들)을 **왼쪽 목록 영역에 끌어다 놓기** — 여러 개 한 번에 가능
- **버튼**: 사이드바 위 **📂 버튼** (또는 대기 화면 "오디오 파일 불러오기")

지원 형식: **wav · mp3 · m4a · aac · wma · flac**

> 불러오면 **목록에만 추가**됩니다(자동 변환 안 함). 변환하려면 목록에서 그 항목을 클릭 → **[텍스트로 변환]**.
> 같은 이름이 있으면 자동으로 `이름 (1)`, `이름 (2)` … 로 구분됩니다.

---

## 5. 변환 모델 고르기 (사이드바 하단)

| 모델 | 특징 |
|------|------|
| **small** | 빠름. 간단 메모·빠른 초안용 |
| **medium** | 중간 |
| **large-v3-turbo** | 정확도 최고. 이 하드웨어에선 medium보다 **오히려 빠름**(권장) |

> 실사용은 **small(빠름) ↔ large-v3-turbo(정확)** 2택이면 충분합니다. medium은 turbo보다 느려 보통 불필요.
> 모델을 바꾸면 **새 변환**부터 적용. 이미 변환된 건 결과 화면 **[다시 변환]** 으로 다시 돌리세요.
> 각 모델은 **처음 그 모델로 변환할 때** 1회 다운로드됩니다(small≈190MB, medium≈539MB, turbo≈574MB).

---

## 6. 실시간 자막

녹음하면서 화면에 자막이 뜨게 하는 기능입니다.

1. 사이드바 하단 **"실시간 자막" 토글을 켬** (한 번 켜면 설정이 저장돼 계속 유지)
2. 녹음 시작 → 화면 아래에 **약 5초마다 자막** 누적

> - 자막은 **선택한 변환 모델**로 나옵니다. **small이 가장 실시간에 가깝고**, medium·large는 정확하지만 **지연될 수** 있어요.
> - 실시간이라 5초 단위로 끊겨 보일 수 있어요(미리보기). 정확한 결과는 정지 후 [텍스트로 변환] 권장.

---

## 7. 결과 화면 (재생 + 텍스트)

목록에서 **변환된 녹음(✓)** 클릭 → 결과가 열립니다.

- **재생**: ▶ 또는 **스페이스바**, 슬라이더로 위치 이동
- **따라 읽기**: 재생 중 문장이 **파랗게 강조**되며 자동 스크롤
- **구간 클릭**: 그 문장 시점부터 재생
- 오른쪽 위: 📁 폴더 열기 · 📋 복사 · 🗑️ 삭제
- 아래: **편집** / **다시 변환** / **TXT 내보내기**

**편집하기**: [편집] → 한 줄이 한 문장(구간)입니다. 고친 뒤 **[편집 저장]**(타임스탬프 유지) 또는 **[편집 취소]**(원래대로 되돌림).

---

## 8. 목록 관리

- **검색**: 사이드바 검색창에 이름 일부 입력
- **이름 수정 / 삭제(1개)**: 목록 항목 **우클릭** → 메뉴
- **+ 새 녹음**: 결과 화면에서 메인으로 복귀

**여러 개 한 번에 삭제**
1. "녹음 목록" 오른쪽 **선택 아이콘**(☑) 클릭 → 선택 모드
2. 지울 항목 **체크** (또는 **[전체 선택]** 으로 한 번에)
3. **[삭제 (n)]** → 확인창 한 번 → 삭제. (**[취소]** 로 선택 모드 종료)

---

## 9. 데이터 / 개인정보

- 녹음·텍스트는 **내 PC에만 저장**, 외부 전송 없음
- 저장 위치: 녹음 `내 문서\Voice2Txt\recordings\`, 설정/목록 `%LOCALAPPDATA%\Voice2Txt\`
- **프로그램을 남에게 줘도 내 녹음·목록은 같이 가지 않습니다** (각자 PC에 따로 쌓임)

---

## 10. 자주 묻는 질문

- **느려요** → 큰 모델일수록 느림. small 사용. (그래픽카드 가속 자동 적용)
- **엉뚱한 문장/반복** → 무음·잡음 구간 환각. 자동으로 어느 정도 걸러지나 완전히 0은 아님.
- **첫 변환이 한참 멈춤** → 모델 첫 로딩만 느리고 이후 빠름.
- **다른 PC에서?** → Win10/11 64비트면 zip 풀고 실행만.
